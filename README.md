# トライポット 業務システム

コアリスHD 13事業会社のうち「トライポット（IT・自社制作モデル）」の業務システム。
本部 (`~/projects/coaris/hq/coaris-main/`) からブリッジAI経由で接続される。

## 構造思想

```
本部 (HQ) = 集計 + 会社発行 + 稟議
  ↑ ブリッジAI
トライポット（この階層）
  ├ 月次 / 週次 / 個人 / 制作
  └ 案件・議事録・提案書・見積書・要件定義
```

## 起動

```bash
cd app
npm install
npm run dev
```

## 事業会社ナビ

- `/monthly` 月次ダッシュボード
- `/weekly` 週次ダッシュボード
- `/home?member=xxx` メンバー個人
- `/production` 制作
- `/approval` 承認（申請側）
- `/deals` 案件管理（データ連鎖の中核）

## 認証

ログインは `/login`（事業会社ごとに独立）。本部とは別認証。

## ブリッジ

本部からの集計取得エンドポイント: `/api/bridge/kpi`（後続実装）
